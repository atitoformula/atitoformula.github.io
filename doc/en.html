The editor lets you type formulas with two purposes: visual representation/printing together with analysis/calculation of actual math. You can use formulas which can be calculated or solved, with formulas which are only for export (LaTeX, image, etc.) including usual text. Your formulas can be definitions of variables and functions, reference each other, so it is possible to write simple programs doing numeric calculations.

The editor is available as android app and web app.

Basic editing

Note that in many cases the editor not only inserts a new symbol but corrects what you’ve typed before.

As you type a lot of buttons on the main panel are changing. Pressing a button repeatedly may adjust what was inserted before. For example pressing a fraction button will make it to include more terms in the numerator. For other buttons just check how they are changing after you press them single time.

Many additional buttons appear on the left and right of the screen as you type. They are called context buttons. For example, when you type a letter then the list of functions appears that start with that letter.

Type the entire name of a symbol in LaTeX and a button will appear to insert it. List of names.

You can input any symbol by typing its unicode hexadecimal number. For example, type "2295" and context button with ⊕ symbol appears.

To type a capital letter press and hold a button.

Among all formulas you typed the editor is searching for symbols that are hard to type again: they are added at right-bottom for fast access.

Export/Import

You can export formulas in several formats.

  • Latex

Well known LaTeX format for visual representation and quality printing of formulas. However, it is not reliable if you want to restore formulas back, not designed to exactly preserve the meaning of formula.

  • Text of formulas (short text format)

Text format represents the mathematical meaning of formula in short way (but not how it looks). Likely to be recognized by many other apps for common types of formulas. Matches Matlab language in a lot of ways but not always. Some of what you type might not be possible to convert to this format. May contain unicode (not ascii) symbols when you use ones.

  • Full data (json)

Will let you restore all your formulas exactly as you see now including some additional data. Can be used only by this editor. Not much suitable for manual viewing/editing. Based on JSON format. In Android, files (*.json) can be opened from other apps (will do import).

  • PDF, Image (png)

Note that formulas may look a bit different than in the app. During import the image is simply added as it is: there is no recognition of formulas in it.

If there is no selection then all the formulas will be exported.

When you copy selection then the editor also makes the copy to the system clipboard. By default it uses text format. In settings it can be changed to latex.

In the web app, you can also copy image of selected formulas by pressing Ctrl+Shift+C. To insert from the system clipboard (copied by another app) use Ctrl+Shift+V. The format of formulas will be determined automatically, otherwise it will be inserted as usual text block.

Use of keyboard

For an android device you can use a physical keyboard, as well as in web application.

When you run the web app on desktop PC/laptop, then the user interface is automatically adjusted for keyboard usage. If you do some action without keyboard, a note may appear on top of the screen with the keyboard shortcut for that action. The same shortcuts are usable in android app also (if you have a connected keyboard), however there is no note to learn them (open web app for that purpose).

You can press a context button (at left side) by pressing Space key several times while holding Ctrl key. You can press a suggestion button (at right side) by pressing Space key several times while holding Ctrl and Shift key. The same is also for recently typed letters at the right-bottom corner (when suggestions are not shown).

Functions

If you type a few letters followed by parentheses it will not mean function call (though looks similar). Type the first letter of function's name or the entire name, a context button appears to replace it by function call. For well known functions the editor may automatically correct it after you type function name followed by parentheses. Function calls will be displayed differently compared to when you just type several variables followed by parentheses. Below they will be called simply as functions.

The editor may not display parentheses in some cases but they appear when you set the cursor inside or after the function. If you want parentheses to be always displayed, then select the function with argument, a context button will appear with the same function having the setting that makes it to always display parentheses.

You cannot edit the name of a function. It may be inconvenient if argument’s expression is big. In this case just select the function and you will see that a button appears to copy the argument only. Click it, type a new function then paste the argument.

To use your own functions you may simply type a letter followed by parentheses, a button appears to replace it with a function call which will have that single letter in its name. To create a function with a longer name select a few letters with empty parentheses after them. Note that single letter functions may be displayed differently compared to multiple letter ones.

To define a function, you create a formula which has this function called with single letter arguments, followed by equals and an expression referencing those arguments.

f(x)=x*x
f(x,y)=x*y

Wherever you use that function it can be calculated or easily replaced by its definition if needed.

Drag and drop

You can easily change the order of multipliers or terms. Just select one or more then drag and drop the selection to a new place. It also works with fractions: you can move elements in and out of a fraction, to another one. It also works with equations which have fractions, you can move elements from the left part of equation to the right one and vice versa. The editor will let you only the moves that are generally mathematically correct.

When you select a part of formula then you may see an “Extract” button. It's for convenience and will let you easily move the selection to a new formula which will be the definition of a new variable. Just type the name of the variable and it will be inserted instead of selection. This is to easily split big formulas to smaller ones, referencing each other via variables’ names.

Adding usual text

You may insert text blocks either to use variables with long names or to write comments. “Text” button lets you do that. It appears when you create a new formula (or you type ‘t’.). Another button is below on the main panel.

A dialog will be shown to edit text and there will be an option to choose whether to ignore this text in math as comment or to process it as variable.

Later you can double click to edit text.

When text is big then a vertical button will appear on the right to let you change the width of the text block. At the right-top of text there is also a button to change alignment.

When you type variables (most of the symbols) they each go to separate text blocks, only digits are automatically combined into a single block: number.  When you type (or select) several symbols then a context button also appears to combine them into a single text block, and it will be processed as a single variable with long name (if no spaces, etc.). Formatting like italic or bold is applied to the entire text block.

In android app in settings you can load a custom font (from https://fonts.google.com/) which will be used for comments only. This option is not available on all devices as it requires google services.

Analysis and transformation of expressions

When you select any part of formula, the editor will start searching among all embedded library formulas that can be applied to the selected expression. It will list all other possible expressions that can follow the original one based on math rules. It also includes solutions to classic math tasks. Some complex calculations will be started only after you select a particular expression.

The list of suggestions may be very long but the best suggestions usually appear amongst the first ones. A suggestion may have a button which lets you see the more detailed information about the library formula which was used.

When the editor cannot find a solution, there is a top-right button to start searching in google.

When you click on suggestion then by default it will be inserted replacing the selected expression. A button will appear which lets you revert the previous action, duplicate the original formula where selection was, then make the replacement in the new formula while keeping the old one the same. Or it may also append the suggested expression to the selected one after the equals sign.

Select a single variable and the editor will suggest more operations related to it: substitution, solving equation, etc. The entire formula will be transformed in this case.

Complex numbers

To make calculations with complex numbers you need to simply include symbol i in your formulas. It can be disabled in settings. There are functions which let you get imaginary and real part of complex number, also argument and modulus (Re(z),Im(z),arg(z),|z|).

Use conj function for complex conjugate.

Trigonometry

All calculations of trigonometric functions are performed in radians. Symbol of degree ° is processed as conversion to radians x / 180 * π, for example 45° will be 0.7853…

To type a degree symbol for any number, first press the power button, then the context button with degree appears.

is processed as 1/sin, not as inverse trigonometric function. Use arcsin for that.

Graphs

During calculation or graphing the editor scans through all of your formulas to see how they reference each other. E.g.

r*c
r=sin(x)
c=cos(x)

If there are duplicate definitions for a variable, an exclamation mark will be displayed.

To see graphs first you need to choose formulas which will be included. When you edit a formula it has a button on the left to include it in graphs. After you select at least one formula, press a button at the top left corner to show graphs. If that formula has an equals sign, then the left expression must be a simple variable name or function like f(x). However, you can graph any inequalities producing values 0 or 1.

Note that if you select a single variable in a formula, the editor will try to solve and build the expression for that variable.

The editor will start analyzing all the formulas that you‘ve included in graph, recursively scanning through all the formulas which define variables used in initial formulas you have included.

While building usual graph the editor expects to find a single variable which doesn't have any formula. It will become a variable for x-axis. Other variables for which you should define some numeric values will become parameters.

Depending on settings, “x” variable may be assumed as the default one. The rest of the variables will become parameters automatically with initial default values.

When graph is opened, for each parameter there will be a button on top. Press it to start changing the value of parameter using a slider. There are two buttons which let you change the range of values used by the slider. If you need to set an exact value then set it in the formula for that parameter.

Graphs of functions with two variables (complex functions)

The editor also lets you analyze functions with two variables. It draws a type of colored graph that is usually used to analyze functions of complex number (represented by two real numbers). But you can use it for any type of function with two variables. E.g.

sin(x)+sin(y)

In this type of graph function's value is represented by color. Red value corresponds to maximum value of function, blue value - minimum value.  Each variable corresponds to each axis.

Minimum and maximum values are calculated only within the displayed area. When you move to another area these values may change significantly but there will always be color distribution between red and blue. The correspondence between function value and color is not fixed initially.  You may need to first find an area which produces suitable minimum and maximum values (no singularity in it), then press the top right “Locked colors” button which will fix colors remembering current minimum and maximum values.

Color distribution between red and blue doesn't always let you easily see how function changes. There is waves mode which lets you split the single red blue wave to multiple ones. In this mode the color will no longer let you see where minimum or maximum is.

There is also a special mode when you have two functions of two variables. In this case one main function will define color distribution between red and blue as explained above. Another second function will define brightness from black to the brightest color. You may set the main function by clicking on it at the bottom left corner.

This mode is enabled by default when you have a function of complex number. E.g.

z=x+i*y
f=(z*z-2)/(1-2*i-z)

Only the formula for “f” should be included in graphs. In this case the main function will be the argument of the calculated complex number “f” and the absolute value will define the brightness. There is a top right button which lets you enable or disable the change of brightness by the second function.

Buttons which lock colors or change waves mode will be applied to the current main function. If you want to tune the distribution of brightness then you need to make the second function as the main one temporarily, after you adjust the distribution of color to what you need, make this function second again.

If you want to graph a function of complex argument, the editor lets you graph only this single function. As a workaround, you can adjust formula and convert complex number to real one (using Re(z),Im(z),arg(z),|z|) and then graph any number of such functions.

3D graph

From version 1.1.6. Android only.

You will see “3D” graph button at right when you graph a function of two variables like described before (initially you see the view above on 3D graph and can change XY-area). When 3D is opened, it is not longer possible to change XY-area, you can only rotate the camera around the center of the displayed surface.

When you graph equations x=…, y=…, z=…  then 3D graph is opened for parametric surface. Interval of change for each variable (parameter) is always from 0 to 1. You can add formulas like u=2πu₁, then u₁ becomes variable, and u goes from 0 to 2π. E.g.

x=v*sin(u)
y=v*cos(u)
z=vu
v=3v₁
u=2πu₁

In the usual case there are two parameters (variables), in the case of a single one the graph will be displayed as a ribbon in 3D space.

Equations may be combined into system for convenience.

Systems of equations

The editor can solve systems of linear equations.

You're not restricted to use only numeric coefficients. If you use multiple variables then the editor will assume that the number of actual variables is equal to the number of equations. The rest of the variables will be considered as part of coefficients. You may see several solutions depending on what actual variables are chosen. Functions like sin(x) can also be considered as variables.

You can write a system in matrix form and convert it to classic form and vice versa.

When you select a system you may notice that a context button appears with the same system but with an additional column with conditions. Such systems are useful during calculations. Condition in the last row is optional.

Matrices

With matrices and vectors (particular case of matrix) the editor will do the similar operation as with numbers, except in this case it does not do numerical calculation but transformations of arbitrary expressions producing a single result matrix. If you define a variable as matrix then all other formulas which include it will be calculated as matrices. However in some cases it may be a single cell matrix. The editor displays the size of the calculated matrix for each formula.

For a single cell matrix it can actually display numeric value.

You may also select any part of formula and the editor will calculate the matrix if some variables are matrices.

You can get the value of matrix's cell by using an index. Type the name of matrix in the lowercase then type the index with numeric value e.g.

M=[4,5,3]
m₁ is 4

If matrix has some dimension with size 1 then the index will be applied to another dimension. If both dimensions are greater than one, then the first digit of the number will be the index of row and the second digit - the index of column. Comma may be optionally used between two numbers.

It is possible to get inverse matrix, or raise matrix to integer power.

To use transposition you can add “T” power to a variable, but for matrix label there is a special context button to do that which appears when you edit a matrix.

If you use your own functions, they can accept a matrix as argument or return any other matrix based on expressions you write. It will be different when you use well-known functions like sin which works only with numeric argument. This function will be applied to each cell of matrix if you pass it as argument, another matrix with the same dimensions will always be returned. The same will be applied to all the common functions which cannot process matrix as argument. If you use your own functions, during matrix calculations they are all expanded i.e. replaced by their definition.

If you need to calculate the cross product of two vectors then use cross function. Symbol cross is used as an optional separator and generally ignored.

Calculus

To use differential (like dx), you should use the function with name “d” (typing d followed by x will not be processed as a differential, the context button with the function appears when you type letter d). When you write a simple argument like x, then it will be displayed simply like dx. You can also write any expression. When you select the function, the editor will try to find derivative for argument expression.

Integrals

When you type letter “i” you will see an integral context button appear.

The ability of editor to solve integrals is limited. If it cannot find solution, you shouldn't assume that there is no one. You can easily pass it to Google search to find solution online.

Each integral is actually a function with the name equal to the integral symbol. The function also has parentheses which are usually not displayed, only when you edit it. Editor expects to find a differential inside each term of the expression.

Position of bounds for definite integral can be changed in settings: property “Tg, ctg, etc. instead of tan, cot, etc.”

When you type an indefinite integral you will also see that a context button appears to make it definite.

Integral argument expression may also contain another integral along with differential (meaning double or multiple integral). Each nested integral must also contain a single differential inside parenthesis.

If you want to calculate definite integral then all the nested integrals must use different variables. If you want to solve indefinite integral then all nested integrals must use the same variable.

You can graph definite integrals. Set the lower bound to some numeric value, and set some variable as higher bound. This variable must be different from the integration variable. It will be used for x-axis. Such a graph is not exact and may be not stable when you change graph area especially in the presence of singularity. It will depend on the lower bound and the area of graphing.

Derivatives

There are three ways to write (or find) derivative. First one is to use differential like it was mentioned before. Second way is to put an expression in parentheses, add power with prime symbol (similar to apostrophe) using a context button which appears after you press the power/exponent button (also apostrophe on keyboard).

(x*sin(x))’

Parenthesis are not required for a single function or variable. Higher-order derivatives are also supported.

The third way is to use the fraction of two differentials (Leibniz's notation).

When you type a single differential d() the editor actually shows a context button with the fraction. The third way is more reliable when you have an expression with multiple variables, otherwise it will be unclear what variable should be used to differentiate.

If you need to increase the order of derivative it's done with the context buttons which appear when you select first-order derivative. You may note that the editor uses a special visual block for higher order derivatives, and it is restricted how you can edit it.

In differential equations you may use the last two ways to denote derivative. The third way is still more reliable to let the editor better understand the task.
